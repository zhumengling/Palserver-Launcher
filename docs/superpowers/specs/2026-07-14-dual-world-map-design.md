# 双版本在线地图设计

## 目标

在线地图提供“旧地图”和“1.0 世界树”切换，默认显示 1.0 世界树地图。两张地图分别使用与各自底图匹配的坐标投影，切换时不会改变玩家原始坐标数据。

## 结构

- `mapConfig.ts` 只负责地图元数据、坐标投影和越界判断，便于独立测试。
- `MapView.tsx` 负责玩家轮询、地图切换和标记渲染。
- `App.tsx` 只挂载地图视图，并删除侧栏中的技术栈文字。
- `App.css` 沿用现有视觉语言，为地图工具栏和分段控件补充稳定布局。

## 行为

- 默认模式为 `v1`。
- 旧地图继续使用现有投影和现有 WebP 底图。
- 1.0 世界树地图使用 palworld.gg 的世界树专用瓦片及其独立投影范围。
- 坐标投影超出 0% 到 100% 时不渲染该玩家，避免把不属于当前地图的玩家错误吸附到边缘。
- 切换按钮使用 `aria-pressed` 表达选中状态。

## 验证

- 单元测试覆盖两种投影、边界与越界坐标。
- TypeScript/Vite 前端构建通过。
- Go 测试与 Wails 桌面构建通过。
- 人工检查两张底图及切换控件布局。
